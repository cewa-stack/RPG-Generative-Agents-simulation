# RPG Generative Agents Simulation

An interactive 2D sandbox environment inspired by "Generative Agents: Interactive Simulacra of Human Behavior".

## Features
- **Local LLM Integration**: Powered by Ollama with Llama-3.1-8B (Q8_0).
- **Generative Behavior**: Agents make autonomous decisions based on environment perception.
- **Unity 2D Engine**: High-performance simulation with custom Tiled maps.

## Requirements
- Unity 6 (6000.3.13f1 or newer)
- [Ollama](https://ollama.com/)
- Model: `llama3.1:8b-instruct-q8_0`

## Setup
1. Clone the repository.
2. Ensure Ollama is running: `ollama run llama3.1`.
3. Open the project in Unity 6.
